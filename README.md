# Omni Plugins

Plugin source for Noctalia Shell (Omni build).

## Adding this source

1. Open **Settings -> Plugins -> Sources**
2. Add a new source:
   - Name: `Omni Plugin`
   - URL: `https://github.com/haritha99ch/omni-plugins`
3. Go to **Settings -> Plugins -> Available** and install any plugin from this source

## Plugins

| Plugin | Description |
|---|---|
| [Omni Overlay](omni-overlay/README.md) | Xbox Game Bar-style gaming overlay with Discord voice, OBS controls, audio and performance widgets |
